# Juego de Ruleta con Desafíos - Onlyne

¡Bienvenido a este juego interactivo de ruleta con desafíos divertidos!

## Cómo jugar
1. Haz clic en "¡Girar Ruleta!" para hacer girar la ruleta.
2. Espera a que se detenga y descubre tu desafío.
3. ¡Completa el desafío y diviértete!

## Hosting Gratis
Para alojar esta página gratis, puedes usar GitHub Pages:

1. Crea un repositorio en GitHub.
2. Sube estos archivos (index.html, style.css, script.js).
3. Ve a Settings > Pages y selecciona la rama main como fuente.
4. Tu página estará disponible en https://tuusuario.github.io/turepositorio/
